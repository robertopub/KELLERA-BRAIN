# KELLERA ROADMAP V1

# FASE 1 - MVP DE NAVEGAÇÃO

Objetivo:

Permitir que uma pessoa com deficiência visual utilize funções básicas do smartphone apenas por voz.

Funcionalidades:

- Detectar desbloqueio do celular.
- Informar contexto atual.
- Abrir aplicativos.
- Voltar.
- Ir para tela inicial.
- Ler tela atual.
- Ler notificações.

Status:

EM DESENVOLVIMENTO

Objetivo:

Funcionalidades:
