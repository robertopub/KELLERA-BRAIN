# BRAIN CORE SPEC

## Objetivo

O BrainCore é o orquestrador principal do KELLERA.

Ele coordena todos os agentes internos.

## Fluxo

1. Receber entrada de voz.
2. Detectar intenção.
3. Consultar contexto atual.
4. Recuperar memória relevante.
5. Planejar ação.
6. Tomar decisão.
7. Executar ação.
8. Gerar narração.
9. Retornar resposta final.

## Agentes utilizados

- VoiceEngine
- IntentAnalyzer
- ContextManager
- MemoryManager
- PlanningAgent
- DecisionAgent
- ActionExecutor
- NarrationAgent
