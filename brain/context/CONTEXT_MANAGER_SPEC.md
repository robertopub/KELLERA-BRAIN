# CONTEXT MANAGER SPECIFICATION V1

# Objetivo

Manter o estado atual do smartphone e da conversa.

---

# Responsabilidades

- Armazenar aplicativo atual.
- Armazenar tela atual.
- Armazenar última ação executada.
- Armazenar último comando do usuário.
- Armazenar contexto conversacional.

---

# Exemplo

Aplicativo atual:

YouTube

Tela atual:

Vídeo reproduzindo.

Última ação:

PLAY_MEDIA

Último comando:

"Toque música"

---

# Informações mínimas

- current_app
- current_screen
- last_action
- last_user_command
- last_response
